# PulseChain Validator Staking Launchpad

**Simple • Safe • Single-file staking tool for PulseChain**

Built specifically for customers of **[ValidatorStore.com](https://www.validatorstore.com/)** — the trusted source for PulseChain validator hardware.

### Features

- Clean PulseChain-themed design
- Connect MetaMask (automatically adds PulseChain network)
- Upload your official `deposit_data-*.json` file
- Shows which validators are **Ready for deposit** or **Already Deposited**
- "Check on g4mm4.io" button for every validator
- One-click deposits (32,000,000 PLS each)
- 100% client-side — nothing is sent to any server

### How to use

1. Download the two files:
   - `index.html`
   - `LogoTransparent.png`
2. Put both files in the same folder
3. Double-click `index.html` (or open it in any browser)
4. Connect MetaMask → Upload your deposit file → Deposit

**No installation required.** Works offline after the first load.

### Safety & Security

- This is a **static HTML file** — your private keys never leave MetaMask
- Every transaction must be manually confirmed in MetaMask
- The code is fully open source (MIT License)
- Transactions are identical to the official PulseChain staking launchpad
